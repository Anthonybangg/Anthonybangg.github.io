MY WEBSITES

Hair/Beauty Salon

This is a Website I built for my parents' hair/beauty salon. The website is full functioning in everyway and was coded from purely HTML and CSS hosted on github. This is by far my favorite website I've built so far because this has saved their business from very expensive hosting costs they previously had. This project also gave my people hope in me that I truly can make it self-taught in this industry, and that alone has made a world of difference.

Ecommerce Store

This is my Ecommerce website that I built on Shopify. I've sold about $10,000 worth of fishing products in revanue on this store. Building this website was the catalyst for why I started to pursue Web Development. I honestly enjoyed the website designing aspect of the business the most and took a lot of pride in it.

Beauty Ecommerce Store

I designed this website for a pretty famous lady in my small hmong community to help her convert more sales for her business. This website was made using Wordpress hosted on Bluehost.
